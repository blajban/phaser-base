# Phaser Base Template
Base repository for Phaser games.

## Getting Started
1. Clone the repository:
    ```bash
    git clone https://github.com/blajban/phaser-base.git
    cd phaser-base
    npm install

2. Start server
    ```bash
    npm start

3. Build production
    ```bash
    npm run build

4. Lint code
    ```bash
    npm run lint

3. Run tests
    ```bash
    npm test
